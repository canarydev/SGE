# Colecciones en Python 3 üêç

Python ofrece diversas estructuras de datos para almacenar colecciones de elementos. A continuaci√≥n, se describen cuatro de las m√°s comunes: listas, tuplas, diccionarios y conjuntos.

## Listas (List) üìã

Las listas son secuencias ordenadas y mutables de elementos. Puedes almacenar diferentes tipos de datos en una lista.

```python
mi_lista = [1, 2, 3, 'Hola']
```

## Tuplas (Tuple) üîÑ

Las tuplas son secuencias ordenadas e inmutables de elementos. Son similares a las listas, pero no pueden modificarse despu√©s de la creaci√≥n. Se pueden crear usando par√©ntesis o sin ellos.

```python
mi_tupla = (10, 20, 30, 'Mundo')
mi_tupla = 10, 20, 30, 'Mundo'
```

## Diccionarios (Dictionary) üìñ

Los diccionarios son colecciones no ordenadas de pares clave-valor. Cada elemento en un diccionario tiene una clave √∫nica.

```python
mi_diccionario = {'nombre': 'Juan de Colonia', 'edad': 25, 'ciudad': 'Pythonburgo'}
```

## Conjuntos (Set) üîÄ

Los conjuntos son colecciones no ordenadas de elementos √∫nicos. No permiten duplicados y son mutables. Se pueden crear usando llaves o la funci√≥n <code>set()</code>.

```python
mi_conjunto = {1, 2, 3, 'Python'}
conjunto_vacio = set()
```

En el caso de crear un conjunto de n√∫meros, al realizar un `print()` de los mismos, aparecer√°n sin duplicados y ordenados de menor a mayor.


## ¬øPuedo crear una lista con list(1,2,3)?

La respuesta es **no**, pero, ¬øpor qu√©?

En realidad la funci√≥n <code>list()</code>, que es un operador de casteo, solo puede recibir un par√°metro, por lo que hay que entregarle un iterable, una tupla o un diccionario.

Por ejemplo, para crear esa lista puedo hacerlo con:

```python
mi_lista = list([1, 2, 3])
mi_lista = list((1, 2, 3))
mi_lista = list(range(1, 4))
```

## Versatilidad de <code>list()</code> y <code>dict()</code>:

- Las funciones <code>list()</code> y <code>dict()</code> pueden tomar un iterable como argumento y devolver una lista o diccionario basado en ese iterable. Por ejemplo, puedes hacer <code>list(range(5))</code> para crear una lista de n√∫meros del 0 al 4.
- Las formas con corchetes no tienen esta funcionalidad incorporada, pero a√∫n as√≠, son m√°s comunes para casos sencillos.
