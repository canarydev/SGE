# UT 9. Desarrollo de componentes IV;
## Desarrollo de modelos avanzados
En esta lección aprenderás. A crear informes personalizados, a generar permisos, conectar a través de otras aplicaciones a Odoo y a editar `odoo.conf`.

También aprenderás a crear wizards (modelos transitorios) que facilitan la ejecución de procesos guiados mediante ventanas emergentes.
